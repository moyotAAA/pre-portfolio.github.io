Infrastructure VPN

×

Contexte

Dans le cadre du télétravail pour 20 collaborateurs, sécurisation des accès distants via OpenVPN sur PfSense et interconnexion de deux sites distants (Siège/Agence) via tunnel IPSec Stormshield.

Schéma Réseau

[Capture d'écran : Topologie VPN PfSense <-> Stormshield]

Extrait Configuration (Firewall Rules)

# Shell / PfSense Config snippet
pass in on $WAN proto udp from any to $WAN_IP port 1194
    keep state label "Allow OpenVPN"
pass in on $OVPN_IF from 10.0.8.0/24 to 192.168.1.0/24
    keep state label "Allow VPN to LAN"

Scripting Active Directory

×

Contexte

Automatisation de la création de comptes utilisateurs à partir d'un fichier CSV lors de la rentrée de 150 nouveaux collaborateurs.

Schéma Réseau

[Capture d'écran : Structure OU Active Directory]

Script PowerShell

$Users = Import-Csv "C:\Temp\nouveaux_users.csv"
ForEach ($User in $Users) {
    New-ADUser -Name $User.Nom `
      -SamAccountName $User.Login `
      -Path "OU=Compta,DC=domaine,DC=local" `
      -Enabled $true
}
